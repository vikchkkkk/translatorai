# Polyglot OCR & Translator 🌍

Проект по дисциплине «Возможности Python». Приложение для автоматического извлечения текста (OCR) с изображений на английском и французском языках с последующим переводом на русский.

## 🚀 Основные функции
- **OCR Engine:** Распознавание печатного текста с использованием трансформеров.
- **Instant Translation:** Автоматический перевод на русский язык через `deep-translator`.
- **Text Analytics:** Модуль подсчета лингвистических метрик (количество слов, символов, индекс сложности).

## 🛠 Технические особенности
- **Модель:** `microsoft/trocr-base-printed` (Hugging Face). Это State-of-the-art решение для распознавания латинских шрифтов.
- **Оптимизация:** Применено кэширование моделей через декоратор `@st.cache_resource`, что позволяет избежать утечек памяти и ускоряет работу приложения.
- **Интерфейс:** Построен на фреймворке **Streamlit** с использованием колоночной верстки и виджетов метрик.

## 📦 Стек
- **Python 3.10+**
- **Transformers (Hugging Face)** — работа с нейросетями.
- **PyTorch** — бэкенд для вычислений.
- **Streamlit** — веб-интерфейс.
- **Deep Translator** — API для перевода.

## 📖 Инструкция
1. Установите зависимости: 
   `pip install streamlit transformers torch pillow deep-translator`
2. Запустите проект: 
   `streamlit run main.py`
